Getting Started with WordPress 101

What is WordPress

WordPress.org is open source software you can use to create a beautiful website, blog, or app.
written in PHP and paired with a MySQL or MariaDB database. Features include a plugin architecture and a template system, referred to within WordPress as Themes.

Why Wordpress

The best part about WordPress is that it's easy to use and flexible enough to make different types of websites. That's the main reason why WordPress has grown so much in popularity. According to a recent survey, WordPress powers 32.3% of all websites on the internet.
